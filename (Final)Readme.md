## **Number Systems**

### **Base 2(Binary)**
*Represents any number using 10 digits (0–9)*

### **Base 8(Octal)**
*Represents any number using 8 digits (0–7)*

### **Base 10(Decimal)**
*Represents any number using 10 digits (0–9)*

### **Base 16(Hexadecimal)**
*Represents any number using 10 digits and 6 characters (0–9, A, B, C, D, E, F)*
